# 位运算

> 只支持整型、字符型数据，直接对二进制位操作。底层开发、寄存器配置、通信校验必备。

---

## 1. 六大位运算符一览

| 运算符 | 含义 | 操作数 | 实际用途 |
|--------|------|--------|----------|
| `&` | 与：两位都为1才得1 | 双目 | 把某个bit清0（置0） |
| `|` | 或：有一位为1就得1 | 双目 | 把某个bit置1（设置） |
| `^` | 异或：两位不同才得1 | 双目 | 两数交换、CRC校验 |
| `~` | 取反：0变1，1变0 | 单目 | 按位反转 |
| `>>` | 右移：低位丢弃，高位补0或补1 | 双目 | 相当于除以2的n次方 |
| `<<` | 左移：最高位丢弃，低位补0 | 双目 | 相当于乘以2的n次方 |

---

## 2. 按位与 & —— 清零某位

规则：两位都是1 → 1，否则 → 0

**核心用法**：把数据中某个bit清0，其余位不变。

```
// 把第3位清0，其余保持
a = a & ~(1 << 3);

// 只保留低4位，高位全清0
a = a & 0x0F;
```

记忆：与0则清0，与1则保留。

---

## 3. 按位或 | —— 置1某位

规则：有一位为1 → 1，否则 → 0

**核心用法**：把数据中某个bit置1，其余位不变。

```
// 把第5位置1
a = a | (1 << 5);

// 低4位全部置1
a = a | 0x0F;
```

记忆：或1则置1，或0则保留。

---

## 4. 按位异或 ^ —— 不相同才为1

规则：两位不同 → 1，相同 → 0

**两大用途**：

### 4.1 不借助临时变量交换两数

```
a = a ^ b;
b = a ^ b;  // 原来的a
a = a ^ b;  // 原来的b
```

原理：异或同一数两次恢复原值（`x ^ y ^ y = x`）。

### 4.2 数据校验（CRC等）

发送端逐字节异或得到校验值，接收端同样运算后对比，判断传输是否出错。

---

## 5. 按位取反 ~ —— 全翻转

0变1，1变0，单目运算符。

```
unsigned char c = 0x55;  // 01010101
c = ~c;                  // 10101010 = 0xAA
```

常配合 `&` 使用来构造"清零掩码"：`a & ~(1 << n)` 把第n位清0。

---

## 6. 移位运算 >> <<

左操作数是被移的数据，右操作数是移动位数。

### 右移 >>

| 数据类型 | 高位补什么 | 名称 | 效果 |
|----------|-----------|------|------|
| 无符号数 | 补0 | 逻辑右移 | 等价于除以2^n |
| 有符号正数 | 补0 | 逻辑右移 | 等价于除以2^n |
| 有符号负数 | 补1 | 算术右移 | 等价于除以2^n（保持符号） |

```
unsigned int a = 16;
a >> 2;  // 16 / 4 = 4

int b = -16;
b >> 2;  // 高位补1，结果仍是负数
```

### 左移 <<

统一规则：最低位补0，最高位丢弃。逻辑左移，等价于乘以2^n。

```
int a = 3;
a << 2;  // 3 * 4 = 12
```

> 注意：左移可能溢出，高位丢弃后值可能不对。

---

## 7. 常见组合操作速查

| 目标 | 操作 | 示例 |
|------|------|------|
| 把第n位清0 | `a & ~(1 << n)` | `a & ~(1 << 3)` |
| 把第n位置1 | `a | (1 << n)` | `a | (1 << 5)` |
| 翻转第n位 | `a ^ (1 << n)` | `a ^ (1 << 2)` |
| 取第n位的值 | `(a >> n) & 1` | `(a >> 4) & 1` |
| 保留低n位 | `a & ((1 << n) - 1)` | `a & 0xFF` |
| 交换两数 | 三次异或 | `a^=b; b^=a; a^=b;` |

---

## 易错提醒

1. **位运算只对整型和字符型有效**，浮点数不行。
2. **右移负数时高位补1**（算术右移），别以为一定补0。
3. **左移溢出风险**——高位丢弃可能导致结果远小于预期。
4. **异或交换法慎用**——同一变量自异或会变成0（`a ^= a` → 0）。
5. **优先级陷阱**——位运算优先级低于算术和关系运算符，复杂表达式多加括号。

---

## 小结

位运算就是直接操作二进制位的工具箱：
- 需要清0 → `& ~mask`
- 需要置1 → `| mask`
- 需要翻转 → `^ mask`
- 需要乘除2^n → `<< n` / `>> n`

嵌入式、驱动开发、协议解析中用得极多，务必熟练掌握组合操作。
