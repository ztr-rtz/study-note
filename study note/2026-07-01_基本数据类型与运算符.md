# 基本数据类型与运算符

C 语言基础的第一章，看起来知识点很碎，但核心就是两件事：数据在内存里怎么存，运算符怎么把数据拼成表达式。

## 1. 数据类型

C 语言的数据类型分两大类：

- 基本类型：整型、浮点型、字符型、void（空类型）、bool（要包含 `<stdbool.h>`）
- 构造类型：结构体、共用体、枚举（后面章节会单独讲）

## 2. 整型

整数在内存里用补码存储。正数的补码就是原码；负数的补码要算：先取绝对值转二进制得到原码，再按位取反得到反码，最后加一。

`sizeof` 是关键字，不是函数，用来算某个类型或变量占多少字节。

常见整型：

| 类型 | 大小 | 有符号范围 |
|------|------|------------|
| short | 2 字节 | -32768 ~ 32767 |
| unsigned short | 2 字节 | 0 ~ 65535 |
| int | 4 字节 | 约 -21 亿 ~ 21 亿 |
| unsigned int | 4 字节 | 0 ~ 42 亿 |
| long | 8 字节（64 位系统） | 很大 |
| unsigned long | 8 字节 | 很大 |

默认的隐式提升方向：`char -> short -> int -> long`。

## 3. 字符型

char 本质上就是小整数，存的是 ASCII 编码。`'a'` 对应 97，只占 1 字节。

- 有符号 char：-128 ~ 127
- 无符号 char：0 ~ 255

常用转义字符：`'\n'` 换行、`'\''` 单引号、`'\\'` 反斜杠。

## 4. 浮点型

- 默认写的小数常量是 double，比如 `3.14`。
- 加 `f` 后缀才是 float，比如 `2.15f`。
- float 占 4 字节，double 占 8 字节。
- IEEE 754 简单记：1 位符号 + 指数位 + 尾数位。

## 5. 常量

整型常量：

- `123` 十进制
- `0123` 八进制，前缀 `0`
- `0x123` 十六进制，前缀 `0x`
- 后缀 `u/U` 表示 unsigned，`l/L` 表示 long，`ul/UL` 表示 unsigned long

浮点常量：

- `3.14` 是 double
- `3.14e10` 或 `3.14E10` 是科学计数法

字符与字符串：

- `'a'` 是 char，占 1 字节。
- `"hello"` 是字符串，占 6 字节，因为末尾会自动加 `'\0'`。

注意区分 `0`（int）、`'0'`（字符，ASCII 48）、`'\0'`（空字符，ASCII 0）。

## 6. 变量

- 先定义，后使用。
- 标识符只能由字母、数字、下划线组成，不能以数字开头。
- 不要和关键字或系统函数重名。
- 变量定义后没初始化，里面存的是随机值，所以最好顺手赋初值。

## 7. 表达式

表达式 = 运算符 + 操作数。每个表达式都有类型，也都有值。

## 8. 运算符

算术：`+ - * / % ++ --`

- `/` 对整数取商，比如 `1/2` 结果是 0。
- `%` 只能用于整数。
- `++i` 先加后用；`i++` 先用后加。浮点数也能自增自减。

关系：`> >= < <= == !=`

逻辑：`&& || !`

赋值：`=`

- 左值必须是变量。
- 左右类型不一致时：浮点转整数会截断小数；整数转浮点会补小数位。
- 大类型转小类型会截断高位。

复合赋值：`+= -= *= /= %=`

- 注意优先级：`a *= 5 + 3` 是先算 `5 + 3`，再乘给 `a`。

逗号：从左到右依次算，整个表达式的值是最后一个。

`sizeof`：计算类型或变量占的字节数。

优先级口诀：

- 括号 > 单目 > 双目 > 三目 > 赋值 > 逗号
- 算术 > 移位 > 关系 > 位运算 > 逻辑

## 9. 类型转换

隐式转换：

- 低精度向高精度转。
- char、short 会先提升成 int。
- float 会先提升成 double。
- 有符号和无符号混算时，有符号会被转成无符号，容易出坑。

强制转换：

```c
int a = 5;
double ret = (double)a / 2;  // a 临时变成 double，但 a 本身没变
```

## 容易踩的坑

- 把负数赋给 `unsigned int`，会变成一个很大的正数。
- 整数除法不四舍五入，直接截断。
- `"hello"` 占 6 字节，不是 5 字节。
- 有符号和无符号混算要当心，结果类型会按无符号算。
