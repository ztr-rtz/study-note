## 1. GPIO 输出模式与输入模式

GPIO（General Purpose Input/Output，通用目的输入输出），单片机引脚的两种工作模式：

| 模式 | 方向 | 作用 | 典型场景 |
|------|------|------|----------|
| **输出模式** | 单片机 → 外设 | 给定引脚电平，控制引脚输出高/低电平 | LED 控制、数码管控制、蜂鸣器控制 |
| **输入模式** | 外设 → 单片机 | 检测引脚电平变化，读取外部信号 | 按键检测、ADC 信号检测、传感器数据接收 |

实际应用举例（车载中控系统）：
- 输出：给传感器发起始信号（500ms 高电平 + 40us 低电平）
- 输入：接收传感器 DATA 信号，检测引脚电平变化

独立按键原理：
- 按键一端接 GND，另一端接 GPIO 引脚
- 初始状态引脚为高电平（置 1）
- 按下时 GND 与引脚短路，引脚变低电平 → 检测到按键按下

## 2. 中断概念

**中断**：CPU 在处理某个任务时，被外界更为紧急的事件打断，要求 CPU 优先去处理这个紧急事件；处理完紧急事件后，再回到之前被打断的地方继续向下执行。

核心要素：
- **中断源**：打断 CPU 执行当前任务的源头/事件
- **中断优先级**：CPU 总是先处理优先级高的中断，处理完后再处理优先级低的中断
- **中断嵌套**：CPU 处理一个中断时再去嵌套另外一个中断任务（51 单片机最多两层嵌套）

---

## 3. 51 单片机中断源类型及个数

51 单片机共有 **5 个中断源**，按优先级从高到低排列：

| 优先级序号 | 中断源 | 说明 |
|-----------|--------|------|
| 0 | INT0（外部中断 0） | 引脚 P3_2 电平变化触发 |
| 1 | TIMER0（定时器 0） | 定时器溢出触发 |
| 2 | INT1（外部中断 1） | 引脚 P3_3 电平变化触发 |
| 3 | TIMER1（定时器 1） | 定时器溢出触发 |
| 4 | UART（串口中断） | 串口通信事件触发 |

相关概念：
- **中断向量表**：本质是一个数组，数组中存放的是中断服务函数的入口地址
- **中断向量**：中断向量表中的标号，通过标号可以访问到对应的中断服务函数

---

## 4. 什么是外部中断

**外部中断**：由引脚电平变化所引发的中断。

51 单片机有两个外部中断：
- **INT0**：对应 P3_2 引脚
- **INT1**：对应 P3_3 引脚

触发方式：
- **下降沿触发**：引脚从高电平 → 低电平的瞬间触发中断
- **上升沿触发**：引脚从低电平 → 高电平的瞬间触发中断

以 INT0（P3_2 引脚）为例，默认下降沿触发：按键按下瞬间引脚从高变低，触发外部中断。

---

## 5. 中断处理流程

从中断源发出请求到 CPU 处理完毕，共 6 步：

```
中断源发出中断请求
        │
        ▼
检查 CPU 是否允许中断（总开关） + 该中断源是否被屏蔽（子开关）
        │
        ▼
比较中断优先级
        │
        ▼
保护现场（保存当前 CPU 寄存器状态）
        │
        ▼
执行中断服务函数（回调函数）
        │
        ▼
恢复现场（恢复之前保存的寄存器状态）
        │
        ▼
返回被打断的位置继续执行
```

两个开关的比喻：
- **总开关**：CPU 是否允许响应任何中断
- **子开关**：某个具体中断源是否允许产生中断

## 6. 51 单片机中的定时器

51 单片机有 **2 个定时器**：
- **Timer0**（定时器 0）
- **Timer1**（定时器 1）

两者都是 **16 位自增型定时器**，内部使用 16 位计数器：
- 计数器最大值：65535（`unsigned short` 范围 0~65535）
- 由 TL（初值低位 8 位）和 TH（初值高位 8 位）组成

定时器的作用：实现精准的定时，外设对时序要求比较高时必须使用。

---

## 7. 定时器工作模式

定时器有两种工作模式（基于计数器位数）：

| 模式 | 计数器位数 | 计数范围 | 特点 |
|------|-----------|---------|------|
| **16 位定时器模式** | 16 位（TL + TH） | 0 ~ 65535 | 范围大，溢出后需手动重装初值 |
| **8 位自动重装模式** | 8 位（仅 TL 参与计数） | 0 ~ 255 | 范围小，溢出后 TH 自动重装到 TL，无需手动干预 |

> 8 位自动重装模式常用于 UART 串口通信的波特率生成。

---

## 8. 定时器配置流程

以 1ms 精准定时为例（晶振 12MHz）：

**计算过程：**

```
晶振频率：12MHz
51单片机实际工作频率 = 12MHz / 12 = 1MHz
每个计数周期时间 = 1 / 1MHz = 1us

目标定时：1ms = 1000us
计数次数 = 1000us / 1us = 1000
定时器初值 = 65535 - 1000 = 64535
```

**配置步骤：**

1. **赋初值**：将计算出的初值写入 TL 和 TH
   - TL = 64535 & 0xFF（低位）
   - TH = 64535 >> 8（高位）
2. **打开计数器**：启动定时器开始计数
3. **按固定频率自增**：计数器每个机器周期 +1
4. **溢出触发中断**：计数器从初值加到 65535 + 1 → 溢出 → CPU 收到定时器中断请求

> 定时时间由两个因素决定：**初值大小** 和 **计数频率**。

## 9. 8 位自动重装载定时器工作原理

**工作原理：**

```
┌─────────┐    溢出(255→0)     ┌─────────┐
│  TH     │ ──────────────────→ │  TL     │
│ (储备值) │     自动重装         │ (参与计数)│
└─────────┘                     └─────────┘
                                   │
                              0→255 循环
```

- **TL** 参与实际计数，从初值开始每个机器周期 +1
- **TH** 作为储备值，不参与计数
- 当 TL 计数到 255 + 1 → 溢出时，**TH 的值自动装载到 TL**，无需手动重写
- 循环往复，实现固定周期的定时

**与 16 位模式的对比：**
- 16 位模式溢出后需要手动重新赋值 TL 和 TH
- 8 位自动重装模式溢出后自动恢复，省去手动操作，适合需要稳定周期信号的场合（如 UART 波特率生成）

## 10. PWM 周期

**PWM**（Pulse Width Modulation，脉冲宽度调制）：让引脚产生方波，使引脚周期性产生电平变化。

**PWM 周期**：一个方波所经历的周期，即从一个上升沿（或下降沿）到下一个上升沿（或下降沿）所经历的时间。

```
  ┌──────┬──────┐
  │      │      │
──┘      └──────┘
  ↑               ↑
  上升沿          下一个上升沿
  │←── 周期 ──→│
```

---

## 11. PWM 占空比

**PWM 占空比**：在一个周期内，高电平所占的时间比例。

```
  ┌────────┐        ┌────────┐
  │  高电平 │        │  高电平 │
──┘        └────────┘        └──
  │← 占空比 →│←  低电平  →│
  │←─────── 周期 ────────→│
```

- 50% 占空比：高电平和低电平时间各占一半
- 占空比越大，高电平时间越长 → LED 越亮、蜂鸣器越响

**应用场景**：LCD 屏幕亮度调节、LED 灯亮度调节、蜂鸣器音量控制

**计算示例**（200Hz PWM，50% 占空比，12MHz 晶振）：

```
PWM 频率：200Hz
PWM 周期 = 1 / 200Hz = 0.005s = 5ms
半周期（50% 占空比）= 5ms / 2 = 2.5ms

分频后频率 = 12MHz / 12 = 1MHz → 每个计数 1us
计数次数 = 2.5ms / 1us = 2500
定时器初值 = 65535 - 2500 = 63035
```

## 12. 有源蜂鸣器与无源蜂鸣器

| 对比项 | 有源蜂鸣器 | 无源蜂鸣器 |
|--------|-----------|-----------|
| **震荡源** | 内部自带震荡源 | 无震荡源 |
| **触发方式** | 通电即可发声 | 需要引脚输出高低电平产生震荡 |
| **声音特征** | 固定频率，持续响 | 可输出指定频率/音调 |
| **控制方式** | 给电就响，断电就停 | 需要 PWM 驱动，频率决定音调 |
| **适用场景** | 简单报警提示 | 音乐播放、多音调提示 |

无源蜂鸣器驱动要点：
- 给定引脚一个周期性高低电平（方波），频率决定音调
- 例如 200Hz/400Hz/600Hz/800Hz/1000Hz 对应不同音调
