# C 语言数组

数组是 C 里最基础的数据结构：一组类型相同、数量固定的数据，内存连续存放。

---

## 一、一维整形数组

### 1. 定义

```c
数据类型 数组名[整型常量];
int array[5];       // 5 个 int
int array2[2 + 3];  // 常量表达式也行
int array3[a + 2];  // ❌ a 是变量，不行
void array4[5];     // ❌ 没有 void 类型的数组
```

三条特性：

- **单一性**：所有元素同一类型
- **连续性**：内存地址紧挨着
- **有序性**：元素有固定顺序

### 2. 访问

用 `数组名[下标]`，下标从 0 开始，范围 0 ~ 容量-1。越界访问的行为不确定——编译器不帮你检查。

```c
int a[5];
a[0] = 10;   // 左值：写入
a[1] = 20;
a[4] = 50;   // 最后一个元素
// a[5] = 60; // ❌ 越界

int b = a[1] + 1;  // 右值：读出
```

数组不能整体赋值，`a = 10` 这种写法不合法。

### 3. 初始化

```c
int a[5] = {1, 2, 3, 4, 5};   // 全部初始化
int a2[5] = {1, 2, 3};        // 部分初始化，剩余自动补 0
int a3[5] = {0};              // 全部置 0，最常用
int a4[] = {1, 2, 3};         // 容量由编译器推断（这里是 3）
int a5[] = {0};               // 只有一个元素，值为 0
// int a6[];                   // ❌ 没容量也没初值，编译报错
```

### 4. 遍历

用 for 循环，写和读都用同一套循环结构：

```c
int a[50] = {0};
int size = sizeof(a) / sizeof(a[0]);

// 写入
for (int i = 0; i < size; i++)
    a[i] = rand() % 1000;

// 读出
for (int i = 0; i < size; i++)
    printf("%3d\t", a[i]);
```

### 5. 倒置

首尾对调，只走一半：

```c
for (int i = 0; i < size / 2; i++) {
    int tmp = a[i];
    a[i] = a[size - 1 - i];
    a[size - 1 - i] = tmp;
}
```

### 6. 冒泡排序（升序）

相邻元素两两比较，大的往后冒。外层每走一轮，最大值归位到末尾。

```c
for (int j = size - 1; j > 0; j--) {       // 外层：控制已排好序的边界
    for (int i = 0; i < j; i++) {           // 内层：相邻比较
        if (a[i] > a[i + 1]) {
            int t = a[i];
            a[i] = a[i + 1];
            a[i + 1] = t;
        }
    }
}
```

### 7. 选择排序（升序）

每次都从后面挑最小的换到前面来。外层走一轮，最小值归位到开头。

```c
for (int j = 0; j < size - 1; j++) {
    for (int i = j + 1; i < size; i++) {
        if (a[j] > a[i]) {
            int t = a[i];
            a[i] = a[j];
            a[j] = t;
        }
    }
}
```

冒泡和选择的区别：冒泡是相邻交换（稳定），选择是跨越交换（不稳定）。两者都是 O(n²)，小数据量随便用。

## 二、一维字符数组

字符数组主要用来存字符串。C 语言里字符串以 `\0` 结尾，分配空间时要多留一个字节。

### 1. 定义和访问

```c
char str[100];       // 最多存 99 个有效字符 + 1 个 '\0'
str[0] = 'h';
str[1] = 'e';
str[5] = '\0';       // 结束标志不能省
```

和整形数组一样，不能整体赋值 `str = "hello"` 是错的。但可以用 `puts(str)` 或 `printf("%s", str)` 整体输出——前提是数组里有 `\0`。

### 2. 初始化

```c
// 全部初始化
char str1[6] = {'h', 'e', 'l', 'l', 'o', '\0'};
char str2[6] = {"hello"};
char str3[6] = "hello";       // 最常用

// 部分初始化：剩余自动填 '\0'
char str4[10] = "hello";      // 后 5 个字节全是 '\0'
char str5[10] = {0};          // 空串，全 0

// 省略容量
char str6[] = "hello";        // 编译器给 6
char str7[] = {'h', 'e', 'l'}; // 不是字符串，没有 '\0'
```

容易搞混的三个东西：
- `0`：数字 0
- `'\0'`：字符串结束标志，内存里存的就是 0
- `'0'`：字符 0，ASCII 码是 48

### 3. 遍历

```c
char str[10] = {0};
int len = 0;

// 逐字符输入，遇到 # 停止
for (int i = 0; i < 9; i++) {
    scanf("%c", &str[i]);
    if (str[i] == '#') break;
    len++;
}
str[len] = '\0';  // 手动补结束符

// 逐字符输出
for (int i = 0; i < len; i++)
    putchar(str[i]);
```

### 4. 常用字符串函数（`<string.h>`）

| 函数 | 作用 | 原型 |
|---|---|---|
| `strlen` | 返回有效字符个数（不含 `\0`） | `size_t strlen(const char *s)` |
| `strcpy` | 字符串复制 | `char *strcpy(char *dest, const char *src)` |
| `strcat` | 字符串拼接 | `char *strcat(char *dest, const char *src)` |
| `strcmp` | 按字典序比较 | `int strcmp(const char *s1, const char *s2)` |

**strlen vs sizeof 的本质区别：**

- `strlen` 是标准库函数，运行时从头数到 `\0`，返回有效字符数
- `sizeof` 是关键字，编译期就算好了，返回数组占的总字节数

```c
char str[100] = "hello";
strlen(str);    // 5
sizeof(str);    // 100
```

**strcpy 的坑：** 目标数组必须装得下源字符串。装不下就是越界，程序可能崩，也可能不崩但逻辑已经错了。

**strcmp 返回值：** 相等返回 0，s1 大于 s2 返回正数，s1 小于 s2 返回负数。它逐字符比较 ASCII 码，遇到不同或 `\0` 就停。`str1 == str2` 比的是地址，不是内容——新手常犯的错误。

### 5. 手写实现

理解了原理，手写版更有助于记牢：

```c
// my_strlen
int my_strlen(char s[]) {
    int len = 0;
    while (s[len] != '\0') len++;
    return len;
}

// my_strcpy
void my_strcpy(char dst[], char src[]) {
    int i = 0;
    while (1) {
        dst[i] = src[i];
        if (src[i] == '\0') break;
        i++;
    }
}

// my_strcat
void my_strcat(char dst[], char src[]) {
    int i = 0, j = 0;
    while (dst[i] != '\0') i++;      // 找到 dst 末尾
    while (1) {
        dst[i] = src[j];
        if (src[j] == '\0') break;
        i++; j++;
    }
}

// my_strcmp
int my_strcmp(char s1[], char s2[]) {
    int i = 0, ret = 0;
    while (1) {
        ret = s1[i] - s2[i];
        if (ret != 0 || s1[i] == '\0') break;
        i++;
    }
    return ret;
}
```

## 三、二维字符数组

存多个字符串时用。本质是「数组的数组」——每个元素又是一个字符数组。

### 定义和访问

```c
char str[3][100];          // 3 个字符串，每个最长 99 字符
str[0]     // 第一个字符串（char[]，可以当数组名用）
str[0][0]  // 第一个字符串的第一个字符
str[2][99] // 最后一个可访问的位置
```

### 初始化

```c
char s1[3][50] = {"hello", "ok", "how"};
char s2[3][50] = {{"hello"}, {"ok"}, {"how"}};
char s3[3][50] = {0};                        // 全置空
char s4[][50] = {"hello", "ok", "123"};      // 行数可省，列数不能省
```

### 字符串数组的倒置和排序

思路和整形数组完全一样，只是操作单元从 `int` 变成了字符串，交换用 `strcpy`：

```c
// 倒置
char t[50];
for (int i = 0; i < size / 2; i++) {
    strcpy(t, str[i]);
    strcpy(str[i], str[size - 1 - i]);
    strcpy(str[size - 1 - i], t);
}

// 冒泡排序（字典序升序）
for (int j = size - 1; j > 0; j--) {
    for (int i = 0; i < j; i++) {
        if (strcmp(str[i], str[i + 1]) > 0) {
            strcpy(t, str[i]);
            strcpy(str[i], str[i + 1]);
            strcpy(str[i + 1], t);
        }
    }
}
```

## 四、二维整形数组

可以用它表示矩阵、图像像素这类二维数据。

### 定义和访问

```c
int a[3][4];       // 3 行 4 列
a[0]               // 类型是 int[]，第一行的一维数组
a[0][0] = 2;       // 类型是 int
```

行、列下标都从 0 开始。遍历用双层循环：

```c
// 写入
for (int j = 0; j < row; j++)
    for (int i = 0; i < col; i++)
        a[j][i] = rand() % 100;

// 读出
for (int j = 0; j < row; j++) {
    for (int i = 0; i < col; i++)
        printf("%d ", a[j][i]);
    printf("\n");
}
```

### 初始化

```c
int a1[2][3] = {{1, 2, 3}, {4, 5, 6}};
int a2[2][3] = {1, 2, 3, 4, 5, 6};    // 等价写法
int a3[2][3] = {1, 2, 3, 4};          // 剩余补 0
int a4[2][3] = {0};                   // 全 0，最常用
int a5[][3] = {1, 2, 3, 4};           // 行数自动算（2）
// int a6[3][] = {1, 2, 3};          // ❌ 列数不能省
```

### 内存布局

不管是一维还是二维，内存里都是线性排列。`int a[2][3]` 的实际存储顺序：

```
a[0][0]  a[0][1]  a[0][2]  a[1][0]  a[1][1]  a[1][2]
```

这就是"行优先"存储。

### 用 sizeof 算行列

```c
int a[2][3];
sizeof(a);        // 24  (2×3×4)
sizeof(a[0]);     // 12  (3×4)
sizeof(a[0][0]);  // 4

int row = sizeof(a) / sizeof(a[0]);          // 2
int col = sizeof(a[0]) / sizeof(a[0][0]);    // 3
```

## 五、三维数组（了解即可）

`int a[2][3][4]` —— 本质还是数组的数组的数组，内存依然线性排列。知道它存在就行，实际开发中二维数组就够用了。

## 速查表

| 操作 | 整形一维 | 字符一维 | 字符二维 | 整形二维 |
|---|---|---|---|---|
| 定义 | `int a[N]` | `char s[N]` | `char s[M][N]` | `int a[M][N]` |
| 容量省略 | `int a[]={1,2}` | `char s[]="hi"` | `char s[][N]={...}` | `int a[][N]={...}` |
| 清空 | `int a[N]={0}` | `char s[N]={0}` | `char s[M][N]={0}` | `int a[M][N]={0}` |
| 遍历 | 单层 for | 单层 for（到 `\0`） | 双层 for | 双层 for |
| 整体操作 | ❌ | ❌（用函数） | strcpy/strcmp | ❌ |
| 交换元素 | `int t` 直接赋值 | strcpy | strcpy | `int t` 直接赋值 |
